EJERCICIO: MCD DE EUCLIDES (RECURSIVO)

DESCRIPCIÓN:
Calcula el Máximo Común Divisor (MCD) de dos números enteros positivos introducidos por el usuario.

CARACTERÍSTICAS:
- Utiliza una función RECURSIVA siguiendo el algoritmo de Euclides.
- Cuenta y muestra por pantalla el número de pasos (llamadas a la función) que fueron necesarios para encontrar la solución.
- Valida que los números sean positivos.
